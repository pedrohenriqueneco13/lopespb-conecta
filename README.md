# LopesPB Conecta

## Descrição do projeto

O projeto **LopesPB Conecta** é uma landing page desenvolvida para a **Distribuidora Lopes PB**, com o objetivo de divulgar produtos, promoções e informações principais da distribuidora de forma simples, organizada, responsiva e acessível.

A página foi criada como uma solução front-end para melhorar a presença digital do negócio, facilitando o acesso do público às informações sobre bebidas, promoções, diferenciais da distribuidora e canal de contato por meio do Instagram oficial.

## Tema escolhido

Criação de um site simples para um pequeno negócio local: **Distribuidora Lopes PB**.

## Diagnóstico inicial da necessidade

A Distribuidora Lopes PB utiliza o Instagram como principal canal de divulgação e contato, porém precisava de uma página web mais organizada para apresentar suas informações de maneira clara e acessível.

A necessidade identificada foi criar uma página simples, direta e funcional, capaz de reunir em um só lugar:

- identificação do negócio;
- tipos de produtos oferecidos;
- promoções e combos;
- diferenciais da distribuidora;
- link direto para o Instagram;
- informações básicas sobre atendimento e contato.

## Público-alvo

O público-alvo do projeto são pessoas interessadas em comprar bebidas para consumo próprio, festas, eventos, encontros familiares ou compras em maior quantidade.

A página também é voltada para clientes que desejam encontrar rapidamente informações sobre produtos, promoções e formas de contato com a distribuidora.

## Objetivo da solução

O objetivo principal do site é facilitar a comunicação entre a Distribuidora Lopes PB e seus clientes, direcionando o usuário para o Instagram oficial da distribuidora.

A ação principal da página é incentivar o visitante a acessar o Instagram **@distribuidora_lopespb**, onde poderá acompanhar novidades, promoções e entrar em contato com o negócio.

## Organização do conteúdo

A página foi organizada nas seguintes seções:

- **Cabeçalho e navegação:** apresenta o nome do projeto e links para as principais áreas da página.
- **Hero:** seção inicial com chamada principal e botão de acesso ao Instagram.
- **Produtos:** apresenta categorias de bebidas comercializadas pela distribuidora.
- **Promoções:** exibe exemplos de combos e ofertas.
- **Diferenciais:** mostra motivos para escolher a distribuidora.
- **Instagram:** reforça o canal principal de contato e divulgação.
- **Contato:** apresenta orientação para o usuário acessar o Instagram.
- **Rodapé:** contém informações acadêmicas, nomes dos integrantes e menção à Universidade Católica de Brasília.

## Decisões de design

A identidade visual foi construída com predominância de cores escuras e destaque em amarelo, buscando transmitir contraste, impacto visual e associação com um ambiente de distribuidora de bebidas.

As decisões principais foram:

- uso de fundo escuro para destacar os elementos visuais;
- cor amarela para botões, detalhes e chamadas importantes;
- cards organizados para facilitar a leitura;
- textos curtos e objetivos;
- botões com chamadas claras para ação;
- layout responsivo para adaptação em diferentes tamanhos de tela;
- rodapé com informações acadêmicas do projeto.

## Uso de HTML semântico

O projeto utiliza elementos semânticos para melhorar a organização do código e a compreensão da estrutura da página.

Foram utilizados elementos como:

- `header`
- `nav`
- `main`
- `section`
- `article`
- `address`
- `form`
- `footer`

A utilização desses elementos ajuda a representar corretamente cada parte da página, evitando o uso excessivo de `div` sem necessidade.

## Estilização com CSS

O CSS foi utilizado em arquivo externo, com variáveis de cor, organização por seções e classes com nomes compreensíveis.

A estilização contempla:

- paleta de cores coerente;
- contraste entre texto e fundo;
- espaçamentos consistentes;
- cards organizados;
- botões destacados;
- responsividade;
- foco visual para navegação por teclado.

## Responsividade

A página foi desenvolvida para funcionar em diferentes tamanhos de tela, incluindo computadores, tablets e celulares.

Foram utilizadas estratégias como:

- `grid`;
- `flexbox`;
- `media queries`;
- unidades relativas;
- menu adaptado para telas menores;
- reorganização dos cards em dispositivos móveis.

## Acessibilidade

O projeto aplica práticas básicas de acessibilidade, como:

- textos alternativos em imagens;
- links e botões com textos claros;
- contraste adequado entre texto e fundo;
- estrutura de títulos em ordem lógica;
- possibilidade de navegação por teclado;
- uso de elementos semânticos;
- destaque visual ao navegar com a tecla Tab.

## Funcionalidade JavaScript

A funcionalidade JavaScript implementada no projeto inclui:

- abertura e fechamento do menu em telas menores;
- alteração de estado visual do menu;
- validação simples do formulário;
- exibição de mensagem de confirmação;
- direcionamento para o Instagram oficial da distribuidora.

Essa funcionalidade foi escolhida por ser simples, coerente com o objetivo da página e adequada ao nível introdutório da disciplina.

## Uso de Bootstrap

O Bootstrap foi utilizado de forma introdutória para auxiliar na organização visual e responsividade do projeto.

Foram utilizados recursos simples, como:

- `container`;
- classes de espaçamento;
- organização responsiva;
- apoio na estrutura visual da página.

O uso do Bootstrap não substituiu o CSS próprio do projeto, mas serviu como apoio para melhorar a organização e facilitar a adaptação da página em diferentes telas.

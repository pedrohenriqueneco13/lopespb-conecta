# LopesPB Conecta

## Descrição do projeto

O **LopesPB Conecta** é um projeto acadêmico desenvolvido para a disciplina de Desenvolvimento Front-end, da Universidade Católica de Brasília — UCB. O site é uma landing page baseada em uma distribuidora real, a **Distribuidora Lopes PB**, com o objetivo de apresentar produtos, promoções e direcionar os clientes ao Instagram oficial do negócio.

## Objetivo

Criar uma página simples, responsiva e acessível para divulgar a Distribuidora Lopes PB, organizar suas principais informações e facilitar o acesso ao Instagram oficial da distribuidora.

## Público-alvo

Clientes interessados em comprar bebidas para festas, eventos, encontros familiares ou consumo próprio.

## Organização da página

A página possui seções de apresentação, produtos, promoções, diferenciais, Instagram, contato e rodapé com informações acadêmicas.

## Design e imagens

A identidade visual utiliza fundo escuro com detalhes em amarelo, buscando contraste e destaque para os botões e cards.

As imagens utilizadas foram geradas com auxílio de Inteligência Artificial, com finalidade acadêmica e ilustrativa.

## HTML, CSS e responsividade

O projeto utiliza HTML semântico, com elementos como `header`, `nav`, `main`, `section`, `article` e `footer`.

O CSS foi feito em arquivo externo, com uso de cores, espaçamentos, cards, botões e responsividade para diferentes tamanhos de tela.

## Acessibilidade

Foram aplicadas práticas básicas de acessibilidade, como textos alternativos nas imagens, contraste adequado, botões claros, títulos organizados e navegação por teclado.

## JavaScript

O JavaScript foi utilizado para o menu responsivo, validação simples do formulário, mensagem de confirmação e direcionamento ao Instagram oficial.

## Bootstrap

O Bootstrap foi usado de forma introdutória para auxiliar na organização e responsividade do layout, principalmente com o uso de `container`.

## Testes realizados

Foram testados os links, o menu responsivo, o formulário, a abertura do Instagram, a adaptação em telas menores e o carregamento dos arquivos CSS e JavaScript.
